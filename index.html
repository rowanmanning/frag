Frag is a fluid, responsive, simple grid system for CSS. Frag is easy to use and highly configurable, you can use it with either plain CSS or Sass.

To get started with Frag, read the documentation:

  • Installing Frag
  • Configuring Frag, and customising to suit your needs
  • Full usage documentation

Frag's source code is hosted on GitHub.

Why Frag?

Frag aims to provide you with a simple grid system, nothing more. I built frag originally because there were not many responsive frameworks out there which did exactly as I wanted them to â€“ old IE support and no extra cruft. I talk a little more about Frag in this post.

Frag is also completely configurable to the point where you could use Frag's mixins to build your own completely new grid. I wanted something more like this to allow for extra flexibility; I don't want to tie you into my chosen widths and column counts.

Legacy

Frag 1.0 supported LESS as well as Sass, and handled things a little differently. The Frag 1.0 site is still available here.
